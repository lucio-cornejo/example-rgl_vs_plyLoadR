STANDUP project

 

Objectives:

Important fact:

In order to adapt the medical treatment, ulcer and wound therapeutic follow-up involve periodic visual examination to monitor the evolution of the wound’s shape and color of the tissues.

How we generate ply files

A partial solution in R

rgl


  • Can R’s most popular 3D library solve our problem?

  • No .

  • rgl is a great tool for creating 3D plots in R, but not for loading complex geometries.

  • The plyLoadR package was created to address the limitations of rgl that we encountered in our project.

plyLoadR

A new widget


  • Created via the htmlwidgets package.

  • Wrapper of JavaScript’s most complete 3D library, Three.js, in order to use it directly from R.

  • Allows for the creation of Three.js scenes, via loading ply files into them.

Open Three.js link in a new window, as soon as this slide is revealed.

plyLoadR’s advantages over rgl

Lagless support for translucent meshes


rgl limitation


- rgl + lightweight translucent geometry = No lag


- rgl + heavy/complex translucent geometry = Lag

plyLoadR’s solution


- Translucent graphics loaded via plyLoadR
generate no lag whatsoever.





Example

Faster rendering of scenes


rgl limitation


- rgl use in Quarto or R Markdown files increases the document’s rendering time.


- Slow loading of websites containing heavy rgl scenes.

plyLoadR’s solution


- plyLoadR significantly reduces rendering time and HTML output size (KB).


- In the previous example shown, after rendering in different documents the rgl and plyLoadR similar scenes, the rendering time decreased from 50 seconds to 10 seconds, and the size of the output HTML page approximately shrank from 18000 KB to 10KB, producing a faster page load.

Greater user interactivity


rgl limitation


- Very limited capabilities for user interaction with the scene.


- Deficient official documentation of such user interaction capabilities.

plyLoadR’s solution


- plyLoadR scenes can be customized with tools from Three.js, the most complete JavaScript 3D graphics library in existence.

Conclusion

plyLoadR is a great tool for:

  • Loading complex geometries, stored as ply files, into a website.

  • Rendering translucent geometries in a 3D scene.

  • Reducing a scene’s rendering time, producing also faster page loading.

  • Creating interactive 3D scenes with diverse tools for user interaction.
